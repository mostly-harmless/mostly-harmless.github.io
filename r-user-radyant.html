At Rady I teach a class titled Research for Marketing Decisions to 2nd year MBA students.

For years I used SPSS. SPSS has many tools my students won't need and defaults that may not apply to applications in marketing.

I wanted to use R but without the steep learning curve. I spent many hours programming a custom analysis menu for R using Deducer. Deducer relies on JGR.

Shiny

What is Shiny?

An open-source R-package created and maintained by RStudio, Inc. First released in November 2012. Shiny intro by Joe Cheng.

A Shiny app

  • ui.r defines the user interface
  • server.r specifies the logic behind the user interface

Killer features (for me)

  • If you know R you can create a UI in Shiny
  • Runs in a (modern) browser (i.e., not IE)
  • Reactivity makes traditional GUIs look clumsy

Applications in education

  • Create and disseminate knowledge
  • Make research results accessible
  • Customize the user-experience (e.g., intermediate steps)

server.R

N = 30
B0 = runif(1, -5, 5)
B1 = runif(1, -5, 5)
x = seq(-3, 3, length = N)
df = data.frame(x = x, y = B0 + B1 * x + rnorm(N))

shinyServer(function(input, output) {

    output$plot = renderPlot({

        plot(y ~ x, data = df)
        abline(input$b0, input$b1, col = "red")

        if (input$showFit) {
            fit = lm(y ~ x, data = df)
            abline(fit, col = "green")
        }

    }, width = 700, height = 500)

})

Naming conventions

  • Suppose mytool is the name of the tool
  • analysis reactive called mytool
  • summary.mytool summary of text output
  • plot.mytool one or more plots
  • ui_mytool describes the UI for the summary and plot views

mytool example

  • Add the line below to www/navbar.html
<li><a href="#" data-value="mytool">My awesome tool</a></li>
  • Then create a file called mytool.R and drop it into the tools-directory

Analysis, summary, and plot functions

mytool <- reactive({
    lm(y ~ x, data = df)
})

summary.mytool <- function(result) summary(result)

plot.mytool <- function(result) {
    plot(y ~ x, data = df)
    abline(input$b0, input$b1, col = "red")
    if (input$showFit) 
        abline(result, col = "green")
}
